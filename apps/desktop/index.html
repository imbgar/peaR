    Install peaR review skills?

    The action buttons — Post review, Distill, Walk-thru, Explain, Video — run peaR's /pr-* slash commands. Those aren't bundled with Claude, so they need to be installed once.

    peaR will write 6 skill files to ~/.claude/skills/. Nothing else is touched, and you can re-run any time.

    The review tiers (Light / Standard / Complex / Ultra) use Claude's built-in /code-review and work without this.
